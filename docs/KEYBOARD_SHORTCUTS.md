# Keyboard Shortcuts Reference

DEViLBOX supports 6 keyboard schemes matching original tracker key layouts.
Switch schemes in **Settings > Keyboard**. The active scheme persists across sessions.

## Available Schemes

| Scheme | Description |
|--------|-------------|
| **FastTracker 2** | FastTracker 2 keyboard layout from ft2-clone source (ft2_keyboard.c). Authentic DOS/PC map |
| **Impulse Tracker / Schism Tracker** | Impulse Tracker keyboard layout from Schism Tracker source (page_patedit.c). Authentic IT  |
| **OctaMED SoundStudio** | OctaMED SoundStudio keyboard layout from official documentation. Authentic Amiga tracker m |
| **OpenMPT** | OpenMPT / ModPlug Tracker keyboard layout from official wiki documentation. Modern Windows |
| **ProTracker** | ProTracker keyboard layout from pt2-clone source (pt2_keyboard.c). Authentic Amiga MOD tra |
| **Renoise** | Renoise keyboard layout from official documentation. Modern tracker/DAW style. |
