# DEViLBOX Coding Guidelines

## Icons and Emojis
- **Never use emojis/emoticons** in the UI or codebase
- Use Lucide React icons (`lucide-react`) for all icons
- Example: `import { Music, Play, Stop } from 'lucide-react'`

## UI Components
- Use the unified Button component from `@components/ui/Button`
- Use design system color tokens (text-accent-primary, bg-dark-bgSecondary, etc.)
- Follow the existing component patterns in the codebase
